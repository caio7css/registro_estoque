# 📦 Sistema de Controle de Estoque em C

## 📌 Sobre o projeto

Bem-vindo ao **Mzr's Marketplace**!
Este projeto consiste em um sistema de controle de estoque desenvolvido em linguagem C, com o objetivo de praticar lógica de programação, estruturas de dados básicas e organização de código.

O sistema segue a ideia de um **CRUD**, com algumas funcionalidades adicionais para melhor controle dos produtos.

---

## 🎯 Funcionalidades

O sistema permite:

* ➕ Cadastrar produtos
* 📋 Listar produtos cadastrados
* 🔍 Buscar produto pelo código
* ✏️ Atualizar informações de um produto
* ❌ Remover produto do estoque
* 📦 Visualizar quantidade disponível

Cada produto possui:

* Código
* Nome
* Quantidade em estoque
* Preço
